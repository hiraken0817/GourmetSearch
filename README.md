# GourmetSearch
【iOS】飲食店検索アプリ

## 開発環境
### 開発環境
Xcode 12.5

### 開発言語
Swift 5.2

## 動作対象端末・OS
### 動作対象OS
iOS 14.3

## アプリケーション機能

### 機能一覧
- 店舗検索：現在地からの距離で店舗を検索できる

### 画面一覧
- 検索画面：現在地からの距離で店舗を検索できる画面
- 結果一覧画面：検索結果を一覧で表示する画面
- 店舗詳細画面：店舗の詳細情報が見られる画面

### 使用しているAPI,SDK,ライブラリなど
- ホットペッパーAPI
- Alamofire
- Nuke
